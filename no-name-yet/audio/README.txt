Drop your audio files here with these exact names (the game loads them
automatically; missing files are simply ignored, so the game still runs):

  shoot.wav      - fired on each shot
  hit.wav        - fired when an enemy dies
  pickup.wav     - fired when an XP gem is collected
  level_up.wav   - fired on level up
  death.wav      - fired when the player dies
  music.ogg      - background music (loops)

Quick way to make SFX: https://sfxr.me (export as .wav)
Free music/SFX: https://freesound.org , https://opengameart.org

For music.ogg, enable looping in the Godot import tab:
  select music.ogg > Import tab > check "Loop" > Reimport.
